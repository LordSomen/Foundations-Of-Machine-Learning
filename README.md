# Foundations of Machine Learning

This repository contains coursework for a Foundations of Machine Learning class. It brings together notebook-based assignments, reports, and datasets that cover core ML concepts such as exploratory data analysis, preprocessing, feature engineering, supervised learning, model evaluation, and visualization.

## What is in this repo

The repository is organized around two assignment sets:

- `CS24MTECH15013-FOML-ASSIGNMENT-1/` includes solutions for earlier ML tasks such as travel data analysis and wine-quality modeling.
- `CS24MTECH15013_Assignment2_FOML/` includes later assignments such as employee burnout prediction and MNIST digit modeling.

## Current projects

### Assignment 1

- `solution-Question-1/` - travel data analysis notebook focused on working with tabular data, cleaning, and model-based experimentation.
- `solution-Question-2/solution_2(c)/` - wine quality notebook that works with red and white wine datasets and applies ML techniques for quality prediction/classification.
- `solution-Question-3/` and `solution-Question-4/` - additional assignment folders for related coursework material.

### Assignment 2

- `Employee_Burnout/` - a burnout prediction project using employee-related features and train/validation/test splits.
- `MNIST/` - a handwritten digit recognition project based on the MNIST dataset.

## Concepts covered

Across the notebooks, the coursework touches on:

- data loading and inspection
- missing-value handling and preprocessing
- feature engineering and transformation
- classification and regression workflows
- model comparison and evaluation
- plotting and result interpretation

## Datasets and artifacts

The repo includes the datasets used in the assignments, along with supporting reports and requirement notes. Most of the work is implemented in Jupyter notebooks, so the notebooks are the best starting point for understanding each solution.

## Running the notebooks

You will typically need:

- Python 3
- Jupyter Notebook support in VS Code or another notebook environment
- common ML packages such as `numpy`, `pandas`, `scikit-learn`, `matplotlib`, `seaborn`, `scipy`, and `statsmodels`

Some folders also include requirement notes that can be used to recreate the intended environment with conda or pip.

## Suggested starting points

1. Open the notebook inside the project you want to study.
2. Read the matching `report.md` or requirement note if available.
3. Run the notebook from top to bottom to inspect the full workflow and results.
